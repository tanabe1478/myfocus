# フィード分類

フィードはSQLiteの`feeds.category`へ任意のカテゴリ名を保存し、サイドバーでグループ表示する。

## 操作

- フィード追加フォームで既存カテゴリを選択、または新しい名前を入力
- フィード行にマウスを置き、分類ボタンからカテゴリ名を変更
- 空欄を入力すると未分類へ戻す
- フィードをカテゴリ行へドラッグして移動
- フィードを「フィード」見出しへドラッグして未分類へ移動
- カテゴリ行の編集ボタンから名前を変更

カテゴリ名変更先が既存カテゴリと同名の場合は、両カテゴリを統合する。所属フィードがなくなったカテゴリは独立したテーブルを持たないため自動的に表示から消える。
